🔷 Traditional For Loops

for (int i = 0; i < rows; i++) {
    for (int j = 0; j < columns; j++) {
        // Access using indices
        table[i][j] = ...
    }
}

✅ Gives you access to indices (i, j)
✅ Can modify array elements

💜 Enhanced For-Each Loops (Student's Code)

for (int[] row : table) {
    for (int value : row) {
        // Direct access to values
        System.out.print(value);
    }
}

✅ Cleaner, more readable
✅ No indices to manage
❌ Cannot modify array elements
❌ No direct index access

📖 How Enhanced For-Each Works:
• for (int[] row : table) - Gets each entire row array one by one
• for (int value : row) - Gets each value in the current row one by one
• The loop automatically knows when to start the next row
• Purple cells show values that have been printed
• Blue cell shows the current value being printed
